# CloudZero-Video-Editor
A lightweight video editor packaged as a simple .exe file

NOTE: My programs do not contain any malicious code. If you get a popup about running my program then it is safe to click "run it anyway"
Please do not send my programs to Microsoft or any other company. Thank you.

File Sizes:
- Windows (.exe): 99.84 MB
- Mac (.zip): 737 MB
- Mac (.app): 635 MB

Note: The program automatically check this repository for updates and will show a popup with a link to the newest release if there is one.
Note: This project is a work in progress and I am still adding validation checks for exporting. Please use the suggested formats for time points.

Instructions:

- 1: On the right side is a button that says "CloudZero-Video-Editor" and "Latest" (written in green) click that
- 2: read the update info
- 3: scroll down and click to download the CloudZero-Video-Editor.exe
- 4: place it in a new folder and run it. An editor_clips file will be generated to place your clips in
- 5: An output folder and a settings.json file will also be generated in the folder
